# Nutripoint Finance

Base inicial da aplicação web que substituirá a planilha de fluxo de caixa.

## Como executar

```powershell
python run.py
```

Depois, abra:

```text
http://127.0.0.1:8000
```

## Usuários iniciais

- Administrador: `admin@nutripoint.local` / `admin123`
- Usuário comum: `usuario@nutripoint.local` / `usuario123`

## Estrutura inicial

- Login com sessão
- Perfis `admin` e `user`
- Painel principal
- Lançamentos
- Fluxo de Caixa
- DRE
- Conciliação
- Administração de usuários
